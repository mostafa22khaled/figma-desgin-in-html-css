# Positivus Landing Page

This project is a **responsive landing page** created based on a **Figma design**.
It is fully implemented using **HTML** and **CSS**, with attention to layout, typography, and responsive behavior.

## Project Overview

The goal of this project was to transform a Figma design into a functional web page. The landing page adapts to different screen sizes, ensuring a smooth experience on desktop, tablet, and mobile devices.

**Figma Design Link:** [Positivus Landing Page](https://www.figma.com/design/FBIeGth6JVstMYnyHkUdbl/Positivus-Landing-Page-Design--Community-?node-id=330-762&t=XF8SE9JkMKLgp1JV-0)

## Features

- Fully responsive layout
- Clean and semantic HTML structure
- Styled with CSS for typography, colors, and spacing
- Sections corresponding to the original Figma design
- Smooth adaptation across devices (desktop, tablet, mobile)

## Technologies Used

- HTML5
- CSS3
- Responsive design techniques (media queries, flexible layouts)
